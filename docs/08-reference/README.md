## 🔍 빠른 참조

### 주요 개념
- **Node**: 그래프의 노드 (파일, 클래스, 함수 등)
- **Edge**: 노드 간의 관계
- **Namespace**: 논리적 파일 그룹
- **Query**: Tree-sitter 기반 분석 쿼리

### API 참조
- `analyzeFile()` - 파일 분석
- `getQueriesForCategories()` - 쿼리 조회
- `runNamespaceAnalysis()` - 네임스페이스 분석
